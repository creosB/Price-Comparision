Program tamamen bana aittir ve sadece eğitim amaçlıdır.Herhangi bir marka ile anlaşma veya zarar verecek unsur bulunmamaktadır.<br>

<blockquote>
<p class="has-line-data" data-line-start="7" data-line-end="8">İçerisinde:<br>
It contains:</p>
</blockquote>
<ul>
<li class="has-line-data" data-line-start="8" data-line-end="9">3 farklı siteden fiyat bilgisi alabilir.<br>
You can get price information from 3 different sites.</li>  
<li class="has-line-data" data-line-start="9" data-line-end="10">Aldığı fiyat bilgilerini hem konsola hem de metin belgesine yazdırır.<br>
It prints the price information it receives both to the console and the text document.</li>
<li class="has-line-data" data-line-start="10" data-line-end="11">Yazdırdığı metin belgesinden karşılaştırma yaparak fiyat değişikliğini görebilirsiniz.<br>
You can see the price change by comparing the text document it prints.</li>
<li class="has-line-data" data-line-start="11" data-line-end="12">Link yazan text dosyalarındaki linkler ile fiyat bilgisini elde eder.<br>
Obtains the price information with the links in the text files that write the link.</li>
<li class="has-line-data" data-line-start="11" data-line-end="12">Main.exe ile başlatabilirsiniz.<br>
You can start it with main.exe.</li>
